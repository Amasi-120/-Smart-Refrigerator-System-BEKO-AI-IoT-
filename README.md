Smart Refrigerator System – BEKO (AI + IoT)
 Project Overview

The Smart Refrigerator System (BEKO) is an AI + IoT based solution designed to track food items, monitor expiration dates,
and send notifications to users to reduce food waste and improve kitchen management.

This project was developed as part of an Artificial Intelligence course.

 Problem Statement

Many households struggle with:

Forgetting food expiration dates
Food spoilage and waste
Poor temperature monitoring inside refrigerators

This system provides an intelligent solution that helps users manage food efficiently and safely.

Solution 

The system uses Artificial Intelligence, IoT sensors, and image recognition to:

Scan food items inside the refrigerator
Monitor temperature and storage conditions
Notify users before food expires
Provide smart recommendations
 Features
 Image recognition to identify food products
 Temperature monitoring using IoT sensors
 Expiration date alerts & notifications
 Mobile connectivity with real-time updates
 Smart dashboard to display food status
 Color indicators for food condition:
Green → Fresh
Orange → Expiring soon
Red → Expired
